## Что такое Tuck в одном абзаце

**Tuck** — это open-source менеджер секретов (secrets manager), написанный на Go и спроектированный «анти-Vault»: один статический бинарь (~20 МБ), без внешней базы данных, без Consul/etcd, с автоматическим распечатыванием (auto-unseal) по умолчанию и встроенной Kubernetes-интеграцией. Развёртывание сводится к `kubectl apply` или `helm install`. По набору возможностей (движки секретов, методы аутентификации, политики, HA) Tuck повторяет ментальную модель HashiCorp Vault, но радикально упрощает эксплуатацию.
